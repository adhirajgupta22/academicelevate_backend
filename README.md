# academicelevate_backend
Backend for a company website that accepts user data such as their name number email and the course they want to buy and sends it to the HR of the company.

### https://academicelevate-backend.onrender.com/
- This is the API it displays Home page on the normal url but when we fo to "API/submit" and do a ## POST request the data can be send to the email of the Receiver.
- To use this API clone this repo, make a .env file then fill up localhost port,set email and put a password key of a particular gmail.com. "Pass key not the Password"
